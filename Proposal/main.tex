\chapter{Introduction}

Financial markets are highly complex systems that are influenced by the behaviour of humans and economic events. Predicting financial time series has always been a big challenge in finance and data science. Traditionally, autoregressive models or technical indicators were used to achieve this goal. While this approach created a mathematical and statistical foundation, it falls short when confronted with unstructured data like news headlines or investor sentiment.

With the rapid advances of artificial intelligence (AI), especially in natural language processing (NLP), new ways for understanding those unstructured texts have been found. For example, transformer models can extract semantic information from text. This information could be used to engineer new features that help explain variance in the movement of stock market indices like the S\&P500 or the MSCI World.

The goal of this master thesis, therefore, is to investigate if a stochastic forecasting approach can be improved by adding sentiment-based features extracted from news headlines and/or social media posts. As a proof of concept (PoC), the work will analyse an ARMA model with and without sentiment integration. Later, more complex AI models – ranging from linear regression and XGBoost to recurrent neural networks (RNN), long short-term memory networks (LSTM), and state-of-the-art transformer models like GPT – will be evaluated and benchmarked. To analyse performance, metrics such as Mean Percentage Error (MPE), Accuracy, and F1-Score will be used.

This investigation contributes to an ongoing debate about the efficient market hypothesis, which states that new information is integrated into prices in a very short time, rendering the information useless for predicting future movements.

\chapter{Research Problem and Goals}
\section{Research Problem}

Despite decades-long research in terms of stock predictions, the prognosis of short-term movements remains a challenge. Classic methods depend solely on previous price movements and technical indicators. However, the influential factors that move the market go far beyond these information. News, Social Media, politics, macro-economic events are examples that could contribute to changes in the stock market as well.

Current studies increasingly work with the analysis of media texts to achieve additional prediction-accuracy, though, the quantitative value of these features is debated. Additionally, the interaction between technical indicators and sentiment features — and their impact on model performance across different AI architectures — remains an open question.
\section{Goals}
The goal of this study is to quantify the benefit of market-sentiment to predict changes in stock prices. Different AI-algorithms will be compared and analysed on which would benefit the most from this new feature.

Specifically this study tries to:

1. create a pipeline to collect and combine news articles. Multiple different datasets that contain american news headlines will be combined. The goal is to create a bigger volume of data to create the sentiment feature.

2. take the data and calculate the sentiment score (1: positiv, -1: negativ, 0: neutral) with different methods and models. There will be experiments on which of the algorithms delivers the best results. For Example a lexicon based approach or a state of the art fine tuned transformer models can be used.

3. compare the difference of sentiment-lag (-day-lag vs 1-day-lag, intra-day-lag) to analyse the decay of explainability over time.

4. implement a pipeline for extracting technical indicators from the stock market prices and combining tehm with the sentiment score. The output should be a feature matrix that the different ai-models can use

5. evaluate model performance using standardized metrics such as Mean Percentage Error (MPE), Accuracy, and F1-Score, for both price change prediction (regression) and market direction classification (up/down).

6. analyze Feature Contribution via statistical tests and model weight analysis

\chapter{Literature Review}
Die Idee Marktstimmung mit technischen Indikatoren zu verbinden ist keine neue Idee. Dies wurde bereits von mehreren Studien durchgeführt \cite{ANBAEEFARIMANI2022108742} \cite{10380556} \cite{9175549}. In \cite{ANBAEEFARIMANI2022108742} wurden beispielsweise Sentimentdaten und gängige technische Indikatoren wie Volatilitätsbänder, Volumen, oder EMA (weighted moving average) verwendet um Forex (Foreign exchange markets) zu predicten. Ähnlich hat \cite{9175549} versucht mittels tagesaggregierten Sentiment und Stock-data (wurde nicht weiter beschrieben) die Aktienkurse von einzelnen Unternehmen wie Apple, Amazon, Microsoft vorherzusagen. Beide dieser Arbeiten haben eine Erfolgreiche Korrelation sowie Performance verbesserung feststellen können. Interessanterweise konnte \cite{ANBAEEFARIMANI2022108742} feststellen, dass das Kombinieren von technischen Indikatoren und sentiment besonder performant ist, was dafür sprich, dass es bestimmte synergien zwischen den Features geben könnte. 

Auch social media post (vor allem twitter oder Financial Hubs) werden in vielen Studien für das Sentiment-Feature herangezogen. Zum Beispiel verwendet \cite{CHOI2024121589} unter anderem Eigenschaften wie (views, likes, dislikes, keywords) um das Price Momentum vorherzusagen. Studien wie \cite{khan2022stock} gehen sogar noch einen schritt weiter und kombinieren news und social media um besimmte Titel vorherzusagen, was die Accuracy robuster gemacht hat.

Diese Studien beziehen sich jedoch nur auf einzelnen Unternehmen beziehungsweise auf den Geldmarkt. Ob und wie viel Marktstimmung hilft in effizienten Märkten wie Aktienindexen Kurse vorherzusagen ist teil dieser Arbeit.

\chapter{Methodology}
Die Grundlage dieser Arbeit bildet ein großer Datensatz aus Zeitungsüberschriften, welche aus verschiedenen amerikanischen Medienquellen stammen. Hierfür werden mehrere amerikanische Datensätze kombiniert (z.~B. Kaggle News Headlines Dataset).
Die Daten werden mittels Pipeline vereinheitlicht, Duplikate werden entfernt und in einer Timeseries zeitlich zugeordnet.

Zusätzlich wird Kurs- und Volumendaten des Zielindex über eine öffentliche API (z.~B. investpy API oder Alpha Vantage) gesammelt. Die Frequenz der Daten (wöchentlich, täglich, stündlich) wird an das Volumen/Granualität des Zeitungsüberschriften-Korpus angeglichen (nur bei ausreichend Daten wäre eine Analyse auf stündlicher Basis möglich).

\section{First prototype}
Im Zuge eines minimalen Experiment konnte mithilfe von ARIMA-Modellen erste Hinweise produziert werden, dass Sentiment einen Mehrwert hat. Hierfur wurde ein auto regressives (AR) Modell mit Moving Average (MA) verwendet. Diese Modelle werden vor allem in der statistik und in ekonomischen Szenarien verwendet. Wichtig hier wurde nicht die oben vorgeschlagene Feature Matrix verwendet, sonder ausschließlich der Preis bzw. Sentiment-Score der letzten N Tage.

Ein ARIMA(p,d,q) Modell hat 3 unterschiedliche Parameter \cite{WikipediaARIMA}:
\begin{itemize}
    \item p => die N letzten Datenpunkte (t-1, t-2, ... t-n)
    \item d => Differenzierungsgrad macht die Änderungen stationär (zum Beispiel von Kursbewegung zu Differenz zum Vortag)
    \item q => die N letzten Regressionsfehler
\end{itemize}
